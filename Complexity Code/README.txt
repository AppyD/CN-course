'complexity_project' contains the bulk of the analytical code used to investigate the Oslo Model in this project.

'plotting' contains the code used to make the figures in the report.

'log_bin_CN_2016' is a module containing methods for log binning data.

'oslo_save_data' contains code for saving the generated Oslo Model data into a file, so that it can be loaded into the plotting module.

'oslo_data_2.p' is a pickle file containing saved data. It is recommended to use this in the plotting module in order to reproduce the results in the report quickly, although the data may be generated manually as well.
